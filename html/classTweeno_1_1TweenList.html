Tweeno::TweenList::TweenList ( )

Create a new tween list.

Tween list are used to have multiple tweens grouped with same properties, updating variables at the same time.

void Tweeno::TweenList::on_update ( std::function< void(std::map< const char *, Tween * > &)>  callback)

Set the update callback.

Set the callback to be called on each update, it will be passed the std::map containing all the tweens, so one could get any value of any tween

using namespace Tweeno;
TweenList* tweens = new TweenList();
tweens->add("x")->from(0)->to(150);
tweens->on_update([](TweenList::list_type& tweens) {
float value = tweens["x"]->value();
});
Parameters
callbackThe callback
